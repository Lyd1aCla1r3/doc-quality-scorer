# Provisioning a Compute Node

This guide explains how to provision a new compute node in the production environment. We use compute nodes to handle the primary application workloads.

## Prerequisites

Before you create a compute node, ensure you have the necessary permissions. You must be an administrator to deploy a compute node.

## Steps to Deploy

1. Log in to the management console.
2. Select the **Compute Nodes** tab on the left sidebar.
3. Click the **Provision New Node** button.
4. Select the standard size for the compute node.
5. Confirm the deployment.

## Next Steps

After the compute node starts up, you can monitor its health in the dashboard. The compute node typically takes five minutes to initialize.
